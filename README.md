## ✨ 功能

## 🚀 快速开始

### 方式1: 直接使用EXE（推荐）

[Releases](https://github.com/ZhuLinsen/daily_stock_analysis/releases)中下载最新exe文件，双击打开使用。



### 方式2: 使用.bat启动项目

双击使用launch.bat，启动项目GUI界面，需要电脑安装python环境。

使用此方式保存的API会在本地生成一个.XX_key用于保存对应服务商的API，如.deepseek_key。
